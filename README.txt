민코주 우리집 도서관 v7 - 기능 크게 확장판

실행:
1) 압축 풀기
2) 터미널에서 폴더 이동
3) npm install
4) npm start
5) http://localhost:3000 접속

기본 관리자:
admin / 1234

유지된 기능:
- 회원가입/로그인
- 관리자 모드
- 책 등록은 관리자만
- 책 등록 폼은 제목/저자만
- 엑셀 파일로 책 일괄등록
- 책 검색
- 대출/반납
- 대출기간 관리자 수정
- 공지 생성
- 연체일수만큼 자동 정지

새로 크게 추가한 기능:
- 예약
- 연장
- 내 서재
- 찜한 책
- 독서목표
- 독서기록
- 희망도서 신청
- 행사 등록
- 회원 수동 정지/해제
- 관리자 통계 대시보드
- 인기 도서
- 새 책 목록
- JSON 백업
- 엑셀 전체 내보내기
- 도서 상태 로그용 DB 구조
- 운영 매뉴얼/샘플 데이터 포함
